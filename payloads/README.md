## Payloads

`render.txt`: Use these payloads in instances where you know you won't be able to pop an alert() or view the console.log(). For example, in a Chrome-embedded application or something that doesn't have access to the window API (like a JCEF desktop application or something).


**Log4j Vectors**
- GET/POST params
- Host Header
- User Agent
- Cookies
- Referer
- X-Forwarded-For
- Path params
